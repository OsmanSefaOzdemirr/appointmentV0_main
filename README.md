# appointmentV0_main

This repository contains a static appointment project. The files have been organized into a simple MVC directory structure without modifying their contents.

```
model/       # currently empty, reserved for future data models
view/        # contains original HTML, CSS, JS and asset files
controller/  # currently empty, reserved for future controllers
```

All original application files are under `view/ddo-AppointmentV0-main` preserving their paths.
